# MobileTicketsIonic

Sistema de **gestão de tickets** desenvolvido com **Node.js**, **MySQL** e **React** no frontend.  
Permite emitir senhas, gerenciar atendimento e visualizar relatórios de tickets.

---

## 📌 Tecnologias utilizadas

- Backend: Node.js, Express, MySQL2  
- Frontend: React  
- Banco de dados: MySQL  
- Controle de versão: Git & GitHub
